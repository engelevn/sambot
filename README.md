# SAMBOT – Chatbot Proyecto El Sótano de Sam

SAMBOT es un chatbot desarrollado en HTML, CSS y JavaScript puro, creado como parte del Taller de Integración de Tecnologías Emergentes y Disruptivas.

Este chatbot se integra al proyecto formativo El Sótano de Sam, una iniciativa de moda sostenible enfocada en la reutilización de prendas y el consumo responsable.

## Objetivo

Diseñar e implementar un chatbot básico que:

- Responda preguntas frecuentes del proyecto.
- Brinde información sobre moda sostenible.
- Explique procesos de envíos, pagos y devoluciones.
- Simule atención al cliente dentro de la interfaz gráfica.


# Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (puro, sin frameworks)

---

# Funcionalidades

- Botón flotante interactivo.
- Menú con opciones intuitivas.
- Submenú sobre moda sostenible.
- Información profesional sobre:
  - Envíos y rastreo
  - Pagos digitales
  - Cambios y devoluciones
  - Soporte
- Respuesta automática al saludo.
- Entrada manual de preguntas.
- Navegación mediante botones.

# Enfoque del Proyecto

SAMBOT no solo cumple una función técnica, sino que está alineado con el propósito del proyecto formativo:

Promover moda sostenible, reducir desperdicio textil y fomentar consumo consciente.
